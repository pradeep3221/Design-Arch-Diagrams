# Sample Messaging Configuration

This document provides a sample configuration for messaging in microservices using RabbitMQ.

## RabbitMQ Example
```yaml
rabbitmq:
  host: localhost
  port: 5672
  username: guest
  password: guest
  exchange: sample-exchange
  queue: sample-queue
```
